# angular

## cli

- angular install

`npm install -g @angular/cli`

- new project

`ng new my-first-project`

- new component

`ng generate c path+name`

- new module 

`ng generate m pat+name --flat --module=app`

- new server 

`ng generate s path+name`

- run live

`ng serve --open`
