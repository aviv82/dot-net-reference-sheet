# typescript

## cli

- initialize npm

`npm init`

- install typescript

`npm install -g typescript`

- check version

`tsc --version`

- initialize tsc

`tsc filename`

- run tsc

`node filename`

## methods

- sort array alphabetically

```ts
list: someData[];

list.sort((a,b)=> a.name.localeCompare(b.name));
```
